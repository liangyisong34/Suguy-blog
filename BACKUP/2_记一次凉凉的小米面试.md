> 1.手写一个防抖函数，还顺便问了下call,apply,bind的区别
```
function debounce(fn,time){
        let timer
        return function(...args){
            if(timer){
                clearTimeout(timer)
            }
            setTimeout(function(){
                fn.apply(this,args)
            },time)
        }
    }
```
